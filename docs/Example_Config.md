# Example Configurations

Configuration is done entirely through the Home Assistant UI editor. Add and configure panels via the **NSPanel HAUI** device page in **Settings → Devices & Services**.

- See the [Panel Configuration](config/panels.md) reference for available panel options.
- See the [Panels Overview](panels/README.md) for details on each panel type.
